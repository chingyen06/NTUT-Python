6. 一元二次方程式

一元二次方程式，aX^2 + bx + c = 0，輸入a, b, c, 求方程式的兩個實/虛根。  
T = math.sqrt(b*b-4*a*c)  
第一個根，x1 = (-b+T)/(2*a)  
第二個根，x2= (-b-T)/(2*a)  

本題須import math  

---------------------------------------------------

輸入說明：  
第一行，輸入a (Integer)  
第二行，輸入b (Integer)  
第三行，輸入c (Integer)  

輸出說明：  
當T>=0，輸出x1, x2為實根，輸出到小數點第一位  
第一行，輸出 print("%.1f” %(x1))  
第二行，輸出 print("%.1f” %(x2))  

當T<0，輸出x1 , x2為虛根，輸出到小數點第一位  
第一行，輸出 print("%.1f+%.1fi" %(x11,x12))  
第二行，輸出 print("%.1f-%.1fi" %(x21,x22))  
若x11或x21為0.0時，則不須輸出正負號  

---------------------------------------------------

Example Input 1  
1  
-2  
1  

Example Output 1  
1.0  
1.0  

---------------------------------------------------
Example Input 2  
1  
2  
2  

Example Output 2  
-1.0+1.0i  
-1.0-1.0i  

---------------------------------------------------
Example Input 3  
2  
0  
2  

Example Output 3  
0.0+1.0i  
0.0-1.0i
